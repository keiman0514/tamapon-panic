# たまぽんパニック

たまぽん食堂から生まれた、料理をテーマにした落ちものパズルゲームです。

料理の猫「たまぽん」が、町から消えてしまった「おいしい気持ち」を取り戻すために、料理で悩んでいる仲間たちを助けながら進んでいきます。

## ゲーム内容

- 落ちものパズル
- ストーリーモード
- 9体の料理キャラクター
- ステージごとの背景
- BGM・ボイス演出
- スマホ・PC対応予定

## ストーリー

ある日、たまぽん食堂の町から  
料理を食べた時の「おいしい！」という気持ちが消えてしまいました。

原因は、町のあちこちに現れた料理の仲間たち。

でも、みんな本当は悪者ではありません。  
料理で失敗したり、さみしかったり、自信をなくしてしまっただけ。

たまぽんは料理の心を集めながら、みんなに笑顔を取り戻していきます。

## コンセプト

料理は、誰かの美味しいのために。  
ゲームは、誰かの楽しいのために。  
たまぽんは、誰かの幸せのために。
